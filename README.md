# Ant
Ant is a basic terminal based text editor.
  
  
##Installation
Clone the project

    $ git clone https://github.com/Akshaydevan/ant.git
    $ cd ant
   
Build

    $ ./build.sh
    $ ant

##Shortcutkeys
- **Open**:
```Press CTRL-O```
- **Save**:
```Press CTRL-S```
- **Exit**:```Press Esc```
